:::note[Regulatory basis]
Drug-device combination products where the device function is principal are regulated under **UK MDR 2002, Schedule 2, Rule 13** (Classification) and **Regulation 7A** (post-Brexit combination product provisions). The medicinal substance component must be assessed for safety and quality as part of the device conformity assessment.
:::

---

## The two regulatory routes

Whether a drug-device combination is regulated as a device or a medicine depends on the **principal mode of action**:

| Principal mode of action | Regulatory route |
|---|---|
| **Device function is primary** (pharmacological/immunological action is ancillary) | Regulated as medical device under UK MDR 2002 |
| **Pharmacological/immunological action is primary** (device element is ancillary) | Regulated as medicinal product under Human Medicines Regulations 2012 |

This page addresses **device-led combination products** — where the device function is the principal action.

---

## Classification: always Class III

Under **Rule 13 of Schedule 2, UK MDR 2002**, any device that incorporates, as an integral part, a substance which:

- If used separately, would be considered a medicinal product within the meaning of the Human Medicines Regulations 2012, **and**
- The substance acts on the body in a way that is ancillary to the device

...is classified as **Class III** regardless of what class the device would otherwise be.

This means: even a device that would otherwise be Class I or Class IIa is elevated to Class III if it incorporates an ancillary medicinal substance.

**Examples:**
- Drug-eluting coronary stent → Class III (stenting is principal; drug release is ancillary)
- Antibiotic-impregnated orthopaedic cement → Class III (structural function is principal; antibiotic is ancillary)
- Heparin-coated central venous catheter → Class III (catheter function is principal; anticoagulant coating is ancillary)
- Silver-eluting wound dressing (with systemic/local pharmacological silver action) → Class III
- Hyaluronic acid filler with lidocaine → Class III (volume restoration is principal; local anaesthetic is ancillary)

---

## MHRA's role in assessing the medicinal component

For Class III devices incorporating a medicinal substance derived from human blood or plasma, or for any device incorporating a substance that would otherwise require a marketing authorisation as a medicinal product, the UK Approved Body must consult MHRA on the medicinal substance component:

- The UKAB submits details of the medicinal component to MHRA
- MHRA assesses the safety, quality, and (where relevant) efficacy of the medicinal component in the context of the combination product
- MHRA provides an opinion to the UKAB
- The UKAB takes the MHRA opinion into account when issuing (or declining to issue) the device certificate

This process ensures that neither the device nor the medicinal component aspects are assessed in isolation.

## Technical documentation requirements

The technical file for a device incorporating a medicinal substance must address:

**Device aspects:**
- Standard technical file contents for Class III devices (see [Technical documentation overview](../technical-documentation/overview))
- Design dossier including the nature and function of the incorporated substance

**Medicinal substance aspects:**
- Chemical/biological characterisation of the substance
- Dose and release profile (if applicable)
- Evidence of pharmacological/immunological effect and its ancillary nature
- Safety data for the substance in the context of the device (local vs systemic exposure)
- Biocompatibility of the substance-device interface
- Compatibility of the manufacturing process with the medicinal component (e.g., sterilisation compatibility)
- Stability of the medicinal component within the device under transport and storage conditions

## Practical examples

### Drug-eluting stents

A drug-eluting coronary stent (e.g., sirolimus-eluting, everolimus-eluting) is the paradigmatic Class III combination device:
- The stent's mechanical function (holding the artery open) is the principal mode of action
- The drug prevents restenosis — this is an ancillary pharmacological effect
- The device is classified Class III under Rule 13 (and would also be Class III under Rule 7 as a long-term intravascular device)
- Full QMS + design dossier assessment by UKAB is required
- MHRA consults on the drug component

### Antibiotic-impregnated orthopaedic cement

- Bone cement provides structural fixation of prosthetic implants — principal mechanical action
- Incorporated antibiotic (e.g., gentamicin, clindamycin) prevents infection — ancillary action
- Class III under Rule 13
- MHRA consultation on antibiotic component required

### Heparin-coated catheters

- Catheter function (vascular access, fluid infusion) is principal
- Heparin coating reduces thrombogenicity — ancillary anticoagulant action
- Class III under Rule 13
- MHRA consultation required
