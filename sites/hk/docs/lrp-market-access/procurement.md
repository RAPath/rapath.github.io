# Public Procurement Strategy

> *HK-unique · DH procurement · Stage C mandate*

## Stage C — 23 March 2026

All **applicable medical devices (AMDs)** procured by the DH must have active MDACS listing from 23 March 2026:
- General: Class II, III, IV
- IVD: Class B, C, D

## Timeline for Compliance

| Action | Recommended Lead Time |
|--------|----------------------|
| Appoint LRP | 6–12 months before deadline |
| Prepare technical dossier | 1–3 months |
| Submit via MDIS | Allow 2–6+ months for review (standard route) |
| Expedited route | Shorter — use if reference-country approval exists |

## Private Hospital Requirements

Stage C applies only to DH public procurement. However, many private hospitals and healthcare institutions in Hong Kong independently require or prefer MDACS listing as proof of regulatory compliance. Early engagement with private hospital procurement departments is recommended to identify any internal listing mandates separate from the Stage C deadline.

AMDs refers to medical devices on the DH procurement framework that fall within the mandated classes (Class II, III, IV for general devices; Class B, C, D for IVDs). Devices used only for research, training, or non-clinical purposes are typically excluded. Confirm device eligibility with the DH procurement team.
