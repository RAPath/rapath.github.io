# Useful Links

## Malaysian Regulatory Authorities

| Organisation | URL | Description |
|-------------|-----|-------------|
| **Medical Device Authority (MDA)** | [www.mda.gov.my](https://www.mda.gov.my) | Primary regulator — guidance, alerts, regulatory information |
| **MyMDA Portal** | [mymda.mda.gov.my](https://mymda.mda.gov.my) | Registration, licensing, and post-market submissions portal |
| **Ministry of Health Malaysia (MOH)** | [www.moh.gov.my](https://www.moh.gov.my) | Parent ministry for health policy |
| **National Pharmaceutical Regulatory Agency (NPRA)** | [www.npra.gov.my](https://www.npra.gov.my) | Regulates pharmaceuticals and cosmetics (not medical devices) |
| **Attorney General's Chambers** | [www.agc.gov.my](https://www.agc.gov.my) | Official Malaysian legislation database |
| **SIRIM QAS International** | [www.sirim-qas.com.my](https://www.sirim-qas.com.my) | Malaysian conformity assessment and testing body |
| **Companies Commission of Malaysia (SSM)** | [www.ssm.com.my](https://www.ssm.com.my) | Company registration |
