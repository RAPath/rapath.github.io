# SISCOMEX & Customs Interface for Medical Devices

:::info BR-unique
**SISCOMEX** (*Sistema Integrado de Comércio Exterior*) is Brazil's integrated foreign trade platform, managed by Receita Federal (customs) and ANVISA, among others. All imports of medical devices into Brazil must be processed through SISCOMEX.
:::

## SISCOMEX process for medical device imports

1. **Importer registers the shipment** in SISCOMEX using the device's NCM code and ANVISA registration number.
2. **ANVISA Import Licence (LI) requested** — ANVISA reviews the import request and grants an LI if the product is registered and the importer holds a valid AFE.
3. **Customs clearance** — Receita Federal processes the customs declaration, applying import duties (II — Imposto de Importação) and other applicable taxes (IPI, PIS/COFINS, ICMS).
4. **Physical inspection** (if required by ANVISA or Receita Federal).
5. **Goods released** — importer takes delivery.

## Import duties and taxes

Medical devices in Brazil are subject to import duties (II) and other taxes. Some device types receive preferential import duty rates or exemptions — particularly devices not manufactured domestically. The NCM code determines the applicable tariff rate.
