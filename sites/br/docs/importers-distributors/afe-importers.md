# AFE for Importers — Peticionamento Eletrônico Steps

Importers of medical devices in Brazil must hold a valid **AFE (Autorização de Funcionamento de Empresa)** specifically for import activities. The AFE is obtained via ANVISA's Peticionamento Eletrônico.

## Required documentation for AFE (Importers)

| Document | Detail |
|---|---|
| CNPJ | Current CNPJ certificate from Receita Federal |
| Articles of incorporation | Contrato social or Estatuto social |
| Proof of address | Current proof of business address (water/electricity bill or lease agreement) |
| Responsável Técnico details | Name, professional council registration number |
| Activity description | Description of medical device types to be imported |
| Floor plan | Simple floor plan of the warehouse/storage area |

## Process

1. Log in to Peticionamento Eletrônico.
2. Select "AFE — Importador de Produto para Saúde."
3. Complete the online form and upload documents.
4. Pay the AFE petitioning fee.
5. ANVISA processes the application (2–4 months typical).
6. State VISA endorsement may be required in some states.
7. AFE issued and published in DOU.

- [RDC 755/2022](https://www.gov.br/anvisa/pt-br/assuntos/produtossaude/produtos-para-saude)
