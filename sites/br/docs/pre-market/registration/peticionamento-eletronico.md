# Peticionamento Eletrônico — ANVISA's Online System

:::info BR-unique
ANVISA's Peticionamento Eletrônico is Brazil's mandatory online submission portal for all regulatory petitions — registrations, AFE applications, post-registration changes, and renewals.
:::

## What you can do

- Submit new product registrations (Notificação / Cadastro / Registro)
- Submit post-registration changes (Petição de Alteração)
- Apply for registration renewal
- Apply for or renew AFE
- Respond to ANVISA deficiency questions (Exigências)
- Track petition status in real time

## Getting started

Create an account in the GOV.BR digital identity portal (gov.br) using CPF (Cadastro de Pessoa Física for individuals) or CNPJ (Cadastro Nacional de Pessoa Jurídica for companies), then link it to your ANVISA account in Peticionamento Eletrônico

- [Peticionamento Eletrônico](https://peticionamento.anvisa.gov.br)
- [ANVISA — Taxas](https://www.gov.br/anvisa/pt-br/assuntos/produtossaude/produtos-para-saude/taxas)

---

:::note Official sources
Verify all information against official [ANVISA](https://www.gov.br/anvisa) sources before making regulatory decisions.
:::

All documents submitted via Peticionamento Eletrônico must be in PDF format (except photographs/diagrams which may be JPG or PNG), with file size not exceeding 50 MB per attachment. All technical documentation must be in Brazilian Portuguese or include certified Portuguese translations. Supporting English-language documents (e.g., peer-reviewed publications, FDA approvals) may be submitted in original language with Portuguese summary.
