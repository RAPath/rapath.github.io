# Clinical Evidence Overview

## Clinical evidence by class

| Class | Requirement |
|---|---|
| Class I | Generally not required for Notificação |
| Class II | Literature-based summary for Cadastro |
| Class III | Clinical evaluation report |
| Class IV | Comprehensive clinical evaluation; may require Brazil-specific clinical data |

## Routes to clinical evidence

1. **Systematic literature review** — structured search and critical appraisal of published data
2. **Substantial equivalence** — demonstrating equivalence to a device with documented clinical data
3. **Clinical investigation in Brazil** — under RDC 10/2015 for novel high-risk devices
