Place the following files in this folder:

  logo.svg        — RAPath NZ navbar logo (SVG, ~32px height)
  favicon.ico     — Browser tab icon (shared across RAPath sites)
  rapath-nz-social.png  — OG image for social sharing (1200×630px recommended)

These can be copied from another site in the monorepo (e.g. sites/au/static/img/)
and the logo is typically shared from packages/theme/static/img/.
