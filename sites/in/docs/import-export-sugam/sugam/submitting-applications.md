# Submitting Applications

All CDSCO import licence and vigilance applications are managed through the **SUGAM portal** at [sugam.cdsco.gov.in](https://sugam.cdsco.gov.in).

See [Submitting Applications on SUGAM](/docs/pre-market/import-registration/sugam-submission) for a step-by-step guide.

> **Official source:** [SUGAM portal](https://sugam.cdsco.gov.in)
