# Imdrf Adoption Table

This page provides links and references to official CDSCO and BIS resources.

> **Official sources:** [CDSCO](https://cdsco.gov.in); [SUGAM portal](https://sugam.cdsco.gov.in); [Bureau of Indian Standards](https://www.bis.gov.in); [IMDRF](https://www.imdrf.org)

India has adopted several IMDRF guidance documents within its regulatory framework administered by CDSCO. Key adopted documents include guidelines on software as a medical device (SaMD), clinical evaluation, and post-market surveillance. The following table summarizes IMDRF adoption status:

| IMDRF Guidance Document | Adoption Status | Applicable Rules |
|---|---|---|
| Software as Medical Device (SaMD): Key Definitions | Adopted | Medical Devices Rules 2017 |
| Clinical Evaluation | Adopted | Schedule IV, Medical Devices Rules 2017 |
| Post-Market Surveillance | Adopted | Rule 10, Medical Devices Rules 2017 |
| Risk Management | Adopted | Quality Management System requirements |

For current and comprehensive information on IMDRF guideline adoption, refer to the CDSCO official website and SUGAM portal.
