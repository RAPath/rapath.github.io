# CDSCO Guidance — By Topic

CDSCO publishes guidance documents, office memoranda, and circulars on the [CDSCO website](https://cdsco.gov.in/opencms/opencms/en/Medical-Device-Diagnostics/). Key guidance topics include:

| Topic | Document type |
|---|---|
| Import licence application requirements | Office memoranda, SUGAM help documents |
| SUGAM portal user guides | Portal guidance |
| UDI implementation guidance | CDSCO circular |
| Vigilance reporting (MDVP) | Schedule 7 guidance |
| Clinical investigation requirements | ND&CT Rules guidance |
| GMP / Schedule 5 guidance | ICMED/QCI documents |
| IMDRF guidance adopted by CDSCO | Adoption notices |
| BIS standards applicable to devices | BIS standards index |

For the most current list, visit [CDSCO — Medical Devices Guidance](https://cdsco.gov.in/opencms/opencms/en/Medical-Device-Diagnostics/).

:::info
CDSCO guidance is not always published as standalone documents. Many requirements are communicated through office memoranda and gazette notifications. Monitor the CDSCO website and [What's New](/docs/whats-new/regulatory-changes) regularly.
:::
