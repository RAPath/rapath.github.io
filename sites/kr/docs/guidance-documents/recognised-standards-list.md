MFDS maintains a recognised standards list identifying which IEC, ISO, and KS standards can be used to demonstrate compliance with technical requirements in Korean submissions.

## Why using recognised standards matters

Using a recognised standard streamlines review — MFDS reviewers can verify conformance against known criteria. Using a non-recognised standard requires additional justification.

## Finding the current list

- On the MFDS website under legal resources
- In MFDS Notifications for specific device types
- Via the MFDS Classification Table (which cross-references applicable standards per product code)

:::caution Confirm the correct edition
The recognised list specifies the edition (or edition range) of each standard. Using an edition not explicitly listed may not be directly accepted without additional justification — confirm the current edition list with MFDS before submission.
:::
