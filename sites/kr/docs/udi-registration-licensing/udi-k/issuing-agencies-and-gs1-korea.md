MFDS has authorised the following **UDI Issuing Agencies** to assign Device Identifiers for UDI-K:

| Agency | Code system | Notes |
|---|---|---|
| **GS1 Korea (GS1 한국)** | GTIN (Global Trade Item Number) | Most widely used; compatible with FDA GUDID and EUDAMED |
| **HIBCC** | HIBC code | For HIBCC members; common in North America |

## Getting a GS1 GTIN via GS1 Korea

1. Become a member of GS1 Korea: [gs1kr.org](https://www.gs1kr.org)
2. Apply for a Company Prefix
3. Assign GTINs to your products

:::tip Reuse existing GTINs
Manufacturers already using GS1 GTINs for FDA or EUDAMED UDI can use the **same GTINs** for UDI-K — no new GTIN needed.
:::
